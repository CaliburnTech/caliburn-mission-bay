# Payload Provider Assets

This directory contains organized assets and documentation for all registered payload providers in the Mission Bay marketplace.

## Directory Structure

Each vendor folder contains:
- `company-overview.md` - Company information and capabilities
- `assets/logos/` - Company logos in various formats
- `assets/banners/` - Banner images for capability modals
- `assets/product-images/` - Product screenshots and interface images

## Current Providers

- **Caliburn** - Maritime Operating System and autonomous vessel technologies
- **Northrop Grumman** - Defense systems and targeting technologies
- **Caliola Engineering** - Secure mesh networking and VPN solutions
- **IBM** - Watson AI and enterprise automation solutions
- **Gecko Robotics** - Robotic inspection and readiness intelligence
- **MythosAI** - Maritime AI and autonomous navigation
- **Anduril Industries** - Autonomous defense systems and AI targeting
- **Raytheon Technologies** - Aerospace and defense systems
- **Lockheed Martin** - Aerospace, defense, and space technologies

## Usage

These assets are used by the admin panel for content management and will be dynamically loaded into the marketplace interface. Each vendor can manage their own assets through the admin interface once implemented.
